# Spots Image Sharing Website

An image sharing website.

## Description

Spots is an image sharing webite that allows potential users to create a profile

and upload images to be shared with other users. This site gives potential users

the ability to create and edit a proifle with their name and profile picture.

Users will be able to upload pictures from a phone, computer, or tablet to be

shown on their profile along with a potenial algorithm.

## Tech Stack

- HTML
- CSS
- Responsive Design
